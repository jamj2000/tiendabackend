Tienda Backend

Frontend en https://tiendafrontend.now.sh

API RESTful

ARTÍCULOS

GET /api/articulos Obtener todos los artículos.
POST /api/articulos Añadir un artículo.
GET /api/articulos/{id} Obtener un artículo.
UPDATE /api/articulos/{id} Modificar un artículo.
DELETE /api/articulos/{id} Eliminar un artículo.

CLIENTES

GET /api/clientes Obtener todos los clientes.
POST /api/clientes Añadir un cliente.
GET /api/clientes/{id} Obtener un cliente.
UPDATE /api/clientes/{id} Modificar un cliente.
DELETE /api/clientes/{id} Eliminar un cliente.
