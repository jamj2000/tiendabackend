Tienda Backend

API RESTful

Código fuente en https://github.com/jamj2000/tiendabackend

Se proporciona la siguiente API:

ARTÍCULOS

/api/articulos    POST
Crear artículo
/api/articulos    GET ALL
Obtener todos los artículos.
/api/articulos/id    GET
Obtener artículo
/api/articulos/id    PUT
Modificar artículo
/api/articulos/id    DELETE
Eliminar artículo

CLIENTES

/api/clientes    POST
Crear cliente
/api/clientes    GET ALL
Obtener todos los clientes.
/api/clientes/id    GET
Obtener cliente
/api/clientes/id    PUT
Modificar cliente
/api/clientes/id    DELETE
Eliminar cliente
